Formulario y entradas

"form" es un formulario, agrupa campos, tiene algunos atributos
"action"
"method"

"input" para que el usuario ingrese texto dentro del formulario con datos personales generalmente. atributos para esta etiqueta:
"type"
"name"
"value"
"placeholder"
"required"

"textarea" es un cuadro que nos permite almacenar bastante texto que no ocupa mucho espacio en la pagina porque es o puede ser escroleable dentro de un cuadro. atributos:
"rows"
"cols"

"buton" es la etiqueta para crear botones. sus atributos son:
"type="submit"" o "button"

Multimedia

"img" es la etiqueta utilizada para insertar imagenes, es una etiqueta "bacia" por lo tanto no tiene cierre
sus atributos son:
"src" donde se coloca la ruta de origen de la imagen, "alt" donde se describe lo que hay en la imagen por cualquier imprevisto y para dar accesibilidad a personas con discapacidad visual, "whidth" se utiliza para definir el ancho de la imagen y "Heigt" para definir el alto de la imagen. aunque el aspecto visual es mejor modificarlo con CSS.

tambien podemos combinar "img" y "a" de la siguiente manera:
a href="https://www.google.com">
img src="https://via.placeholder.com/150" alt="Ir a Google">
/a

"audio" audio embebecido. atributos:
"src"
"controls"

"video" video embebecido, atributos: "src"
"controls"
"whidth"
"Heigt"

Scripts y estilos

"link" es una hoja de estilo externa. atributos:
"rel="stylesheet"
"href"

atributos

"id" se utiliza en cualquier elemento, le da una identidicacion unica a cualquier elemento, es muy util a la hora de dar funcionalidades o estilos a los elementos individualmente

"class" se utiliza en cualquier elemento, mayormente se le da la misma clase a un grupo de elementos para dar estilo o funcionalidad en serie a varios elementos, ayuda a repetir menos el codigo CSS o js

"title" aplica a casi todos los elementos, su funcion es mostrar el titulo que pongamos al elemento cuando colocamos el cursor encima de este

"alt" se utiliza en la etiqueta "img" para describir la imagen que aloja la etiqueta

"href" en la etiqueta "a" para definir el enlace o direccion de un archivo

"src" en las Etiquetas "img", "script" y "video" especifica la fuente del archivo

"target" en la etiqueta "a" sirve para abrir el enlace en una nueva pestaña del navegador

Comentarios

empieza con "" se utiliza para comentar el codigo y sea mas facil de comprender o obtener contexto para el programador o grupo de programadores, sirve solo para eso porque no es visible en el sitio wew que se esté creando, aunque si se ve en el codigo fuente. tambien sirve para desactivar algunas partes del codigo para hacer pruebas sin borrarlo.
los comentarios no deben ir dentro de elementos como, por ejemplo "p" porque puede romper el codigo

Estructura semantica

"header" Encabezado general del sitio o de una sección. Suele contener el logo, título o menú.

"nav" Área de navegación (menú principal o enlaces internos).

"main" Contenido principal único de la página. Solo se usa una vez por página.

Agrupa contenido relacionado dentro de una página.

"article" Contenido independiente: una noticia, una entrada de blog, etc

"aside" Información secundaria: barras laterales, enlaces relacionados.

"footer" Pie de página: créditos, contacto, enlaces legales, etc.
