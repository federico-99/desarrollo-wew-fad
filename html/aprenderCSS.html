herramientas css

Inicio

  • 🔹 Nivel Básico: Fundamentos
    • ¿Qué es CSS y cómo se aplica? Introducción al lenguaje, tipos de hojas de estilo (inline, interna, externa).
    • Selectores básicos: uso de selectores por etiqueta, clase, ID, agrupaciones.
    • Sintaxis de reglas CSS: propiedades, valores, llaves {}, y estructura de una regla.
    • Colores y unidades: nombres de color, HEX, RGB, HSL. Unidades como px, %, em, rem.
    • Modelo de caja (Box Model): padding, border, margin, contenido, y comportamiento de los elementos.
    • Propiedades de texto: fuente, tamaño, color, alineación, espaciado entre letras y líneas.
    • Fondos y bordes: colores de fondo, imágenes, tamaños, bordes redondeados y estilos.
    • Display y visibilidad: block, inline, none, inline-block, visibility.
    • Estilos de enlaces y listas: pseudo-clases como :hover, personalización de listas (list-style).
    • Introducción a responsive con media queries simples: cómo cambiar estilos según el tamaño de la pantalla.
  • 🟡 Nivel Intermedio: Diseño y Organización
    • Selectores avanzados: combinadores (>, +, ~), atributos ([type="text"]), pseudo-clases como :nth-child().
    • Herencia, cascada y especificidad: cómo se aplican los estilos y qué tiene prioridad.
    • Flexbox (Flexible Box Layout): diseño unidimensional, alineación, distribución, dirección de los elementos.
    • Grid Layout (CSS Grid): diseño bidimensional, filas, columnas, áreas, auto-placement.
    • Media Queries avanzadas: breakpoints múltiples, orientación, resolución, condiciones combinadas.
    • Transiciones y animaciones básicas: transition, transform, @keyframes, efectos simples al pasar el mouse.
    • Organización del CSS: comentarios, orden lógico de propiedades, convenciones como BEM.
    • Variables CSS (--nombre): reutilización de valores, :root, uso con var().
    • Positioning: static, relative, absolute, fixed, sticky, y z-index.
    • Zonas de scroll y overflow: manejo de contenido desbordado con overflow, scroll y hidden.
  • 🔵 Nivel Avanzado: Interactividad y Rendimiento
    • Animaciones complejas con @keyframes: encadenamiento, control de tiempos, animaciones múltiples.
    • Clases dinámicas con pseudo-clases: :not(), :is(), :has() (con soporte moderno), efectos complejos.
    • Funciones y cálculos: calc(), clamp(), min(), max(). Valores dinámicos y adaptativos.
    • Contenedores de layout modernos: container queries, subgrid y sus aplicaciones.
    • Modo oscuro y temas con CSS: prefers-color-scheme, uso de variables CSS para temas claros/oscuros.
    • Accesibilidad en CSS: buen contraste, foco visible, :focus-visible, prefers-reduced-motion.
    • Optimización de rendimiento: reducción de reflows, uso inteligente de clases, carga crítica de estilos.
    • Preprocesadores (Sass, Less): introducción, anidamiento, mixins, funciones, herencia.
    • CSS Modules / CSS-in-JS (React, Vue): cómo aislar estilos en proyectos con frameworks modernos.
    • Custom properties + JavaScript: cambiar estilos dinámicamente desde JS usando variables CSS.
